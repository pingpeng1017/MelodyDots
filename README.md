### 프로젝트 개요

- 프로젝트 기간: 2023.06.30. ~ 2023.08.10.
- 프로젝트 이름 및 제목: MelodyDots - 시각 장애인을 위한 음악 악보 자동 점자 변환 시스템
- 프로젝트 설명 및 소개:

  음악 점자 변환 프로그램은 시각 장애인을 위한 음악 접근성을 개선하기 위한 프로젝트입니다.
  <br>이 프로그램은 악보 이미지를 입력으로 받아서 음악 점자로 변환해주는 기능을 제공합니다.
